# 📌 Overview

**FixIt** is an AI-powered civic issue detection and reporting platform that aims to help identify public infrastructure problems such as potholes, garbage accumulation, damaged roads, water leakage, and other civic issues using **Computer Vision** and **Artificial Intelligence**.

The objective is to enable citizens to quickly report issues through camera input while providing intelligent analysis that can assist municipal authorities in faster decision-making and resolution.

This project is currently under development, with several advanced features planned for future releases.

# ⚙️ Installation

## Clone the repository

```bash
git clone https://github.com/ryanbasic1/fixit.git
```

```bash
cd fixit
```

## Create a virtual environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux/Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

## Install dependencies

```bash
pip install -r requirements.txt
```

## Run the project

```bash
python main.py
```

or

```bash
python app.py
```

# 🧠 Planned Workflow

1. User captures/uploads an image or video.
2. AI processes the input using Computer Vision.
3. Civic issues are detected and classified.
4. The system generates insights and recommendations.
5. Reports are stored and displayed through the dashboard.
6. Future versions will integrate location mapping and automated complaint generation.
